Boost C++ Libraries
~~~~~~~~~~~~~~~~~~~

These libraries are used in IsoAgLib at least for
  tools/vt2iso
.

This repository contains the minimal subset of the libraries' files
needed for IsoAgLib in the subdirectory named
  boost_1_38_0
.

These files are extracted from
  http://switch.dl.sourceforge.net/sourceforge/boost/boost_1_38_0.tar.gz
.
