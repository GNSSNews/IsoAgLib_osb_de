To properly compile vt2iso within Windows you will need to fulfill the following
requirements (depending on the used platform):

Dev-Cpp
=======
The following packages need to be added to your Dev-Cpp installation (just use
Dev-Cpp's built-in package manager):
- Xerces-C++ v2.5.0.122 (xerces-c_2_5_0-mingw.DevPak)
- FreeImage 3.10.0 (FreeImage.DevPak)
NOTE: DO NOT SET "-O3" AS C++ COMPILER SWITCH OR VT2ISO WILL CRASH!

MS Visual Studio (>= Express 2005)
==================================
Get the following projects and extract them properly(!) to "../libs/":
- Xerces-C++ v2.5.0.0 
  http://archive.apache.org/dist/xml/xerces-c/Xerces-C_2_5_0/xerces-c_2_5_0-windows_nt-msvc_60.zip

- FreeImage 3.10.0
  http://downloads.sourceforge.net/freeimage/FreeImage3100Win32.zip

- Microsoft Platform SDK (Windows® Server 2003 R2 Platform SDK Full Download)
  http://www.microsoft.com/downloads/details.aspx?FamilyId=484269E2-3B89-47E3-8EB7-1F2BE6D7123A&displaylang=en
  

ATTENTION:
The xerces-c_2_5_0.dll that was included with the Dev-Cpp package is NOT the
same as the one you can download from apache.org!! Therefore you can get an
inproper procedurecall messages when starting vt2iso using the wrong dll!
